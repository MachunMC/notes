# 一、操作系统的启动

启动顺序：

BIOS  --->  BootLoader  ---> OS



系统调用：应用程序主动向操作系统发出服务请求

异常：来源于不良的应用程序，非法指令或其他坏的处理状态

中断：来源于外设，来自不同的硬件设备的计时器和网络的中断



系统调用、异常、中断，三者的区别和联系？

**区别**：

- 源头
  - 中断：外设
  - 异常：应用程序意想不到的行为
  - 系统调用：应用程序请求操作系统提供服务
- 处理时间
  - 中断：异步
  - 异常：同步
  - 系统调用：异步或同步
- 响应状态
  - 中断：持续，对应用程序是透明的
  - 异常：杀死或重新执行，产生异常的应用程序
  - 系统调用：等待和持续
